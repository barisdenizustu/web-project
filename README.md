## 🚀 Task Summaries

### Task 2: Git Version Control
I used Git as a Distributed Version Control System (DVCS) to manage this project. The workflow includes repository initialization, staging, and committing. I also explored Git's internal object structure, such as **blobs** for file data, **trees** for directory structures, and **commits** for snapshots.

### Task 3 & 4: Static HTML & Responsive Layouts
"The Book Corner" is built using semantic HTML5. I prioritized accessibility using `alt` attributes and organized the content with `<header>`, `<nav>`, `<main>`, and `<section>` tags. The design follows a **Responsive Layout** approach, ensuring the site is usable on desktops, tablets, and smartphones by using CSS flexbox/grid and media queries.

### Task 5: JSON Data Handling
I organized the bookstore's data using JSON (JavaScript Object Notation). This format was chosen because it is lightweight, easy to read for both humans and machines, and serves as the industry standard for data exchange in web applications.

### Task 6: Security Assessment
I conducted a security evaluation for a P2P marketplace, focusing on four critical areas:
1.  **Access Control:** Implementing Role-Based Access Control (RBAC).
2.  **OWASP Risks:** Mitigating Broken Authentication, Sensitive Data Exposure, and IDOR.
3.  **Server Management:** Secure port configuration and firewalls.
4.  **DoS/DDoS Protection:** Using rate limiting and traffic monitoring to ensure availability.
